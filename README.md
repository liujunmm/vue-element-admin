# vue-element-admin

## 初始化项目

```bash
# 克隆项目
git clone https://github.com/liujunmm/vue-element-admin.git

# 进入项目
cd vue-element-admin

# 安装依赖
yarn install

# 启动项目
yarn run dev
```

本地开发地址： http://localhost:9528

## 构建项目

```bash
# 构建项目（测试环境）
yarn run build:stage

# 构建项目（生产环境）
yarn run build:prod
```

## Advanced

```bash
# 预览发布环境效果
yarn run preview

# 预览发布环境效果+静态资源分析
yarn run preview -- --report

# 代码格式化检查
yarn run lint

# 代码格式化检查并自动修复
yarn run lint -- --fix
```
## 附加说明
```bash

# 全局安装plop
yarn add -g plop

# 自动新建项目模板（视图文件view、组件components、store）
yarn run new
```
